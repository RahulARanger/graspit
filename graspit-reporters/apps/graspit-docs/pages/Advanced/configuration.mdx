## Options

you would see this object in your `attachReporter` function, this is one way to send to your function the required options to the reporter.
some of the options are stored in the database so graspit server can use it or the dashboard can use it or for the reporter that was attached.

<Space style={{ margin: "12px" }} align="baseline">
	<>
		### port _(number)_
		<Tag color="orange">Graspit Server</Tag>
	</>
	If you want graspit server to use this particular port instead of 6969, you could
	this option.
</Space>

<Space style={{ margin: "12px" }} align="baseline">
	<>
		### root _(string)_
		<Tag color="orange">Graspit Server</Tag>
	</>
	Mention the Path where you want to store the folder where you would have the
	database.
</Space>

<Space style={{ margin: "12px" }} align="baseline">
	<>
		### collectionName _(string)_
		<Space direction="vertical">
			<Tag color="orange">Required</Tag>
			<Tag color="orange">Graspit Server</Tag>
			<Tag color="orange">Graspit Dashboard</Tag>
		</Space>
	</>
	Mention the name of the folder where you would have in the [root](#root) folder.
</Space>

<Space style={{ margin: "12px" }} align="baseline">
	<>
		### projectName _(string)_
		<Space direction="vertical">
			<Tag color="orange">Required</Tag>
			<Tag color="orange">Graspit Server</Tag>
			<Tag color="orange">Graspit Dashboard</Tag>
		</Space>
	</>
	Assign a Name for your Project, this would be helpful in dashboard in case if
	you have multiple test configurations.
</Space>

<Space style={{ margin: "12px" }} align="baseline">
	<>
		### timeout _(number)_
		<Tag color="orange">Reporter</Tag>
	</>
	max amount of time an item can remain in the queue before acquiring the lock.
	Reference: [async-lock](https://github.com/rogierschouten/async-lock#options)
</Space>

<Space style={{ margin: "12px" }} align="baseline">
	<>
		### addScreenshots _(boolean)_
		<Space direction="vertical">
			<Tag color="orange">Reporter</Tag>
			<Tag color="orange">Optional</Tag>
		</Space>
	</>
	Reporter specific, it would use [addScreenshot]() function to add screenshot
	if it's [supported]().
</Space>
